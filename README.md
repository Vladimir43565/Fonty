🎯 What is Fonty?

Fonty is a lightweight app that lets you install custom fonts into Microsoft Word instantly—without dealing with Windows font folders or settings.

Just select a font and it appears in Word.

✨ Features
⚡ One-click install
🔒 No admin required
🔄 Auto-restart Word (optional)
🎨 Clean modern UI
🧠 No system reboot needed
🎥 How it works
Open Fonty
Select a .ttf or .otf file
Click install
Done — font appears in Word
